# Инструкция по запуску сайта "Контракт - СВОим" локально

Поскольку сайт современный и использует React (фронтенд) и Node.js (бэкенд для заявок), его нельзя просто открыть как файл. Нужно выполнить эти шаги:

### 1. Подготовка
1. Установите **Node.js** (скачайте с официального сайта nodejs.org, версия LTS).
2. Скачайте проект из Replit: Нажмите на название проекта вверху слева -> **Download as ZIP**.
3. Распакуйте архив и откройте эту папку в **Visual Studio Code**.

### 2. Установка зависимостей
Откройте терминал в VS Code (меню Terminal -> New Terminal) и введите:
```bash
npm install
```
*Это скачает все библиотеки (React, Express, Nodemailer и др.), необходимые для работы.*

### 3. Настройка почты (Опционально)
Чтобы заявки реально приходили на почту:
1. Создайте в корне папки файл с названием `.env`.
2. Добавьте туда строку:
   ```env
   SMTP_PASSWORD=ваш_пароль_от_почты
   ```
   *(Если файла нет, сайт будет работать, но письма будут выдавать ошибку в консоли сервера).*

### 4. Запуск
В том же терминале введите:
```bash
npm run dev
```

### 5. Просмотр результата
После запуска в терминале появится ссылка, обычно:
- Сайт: `http://localhost:5000`
- Админка: `http://localhost:5000/admin-requests-9955`

---
**Почему у вас был белый экран?**
На скриншоте видно, что файл открыт через `Live Server` как `index.html`. В React-приложениях `index.html` — это просто пустая оболочка, которая заполняется кодом только после правильного запуска через `npm run dev`.
